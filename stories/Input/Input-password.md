# Input

Importación del componente Input.

```js
import { Input } from 'library/components/Input';
```

<!-- STORY -->

#### Uso

```jsx
<Input
  className="c-input"
  type="password"
  name="input-password"
  width="290px"
  onChange={e=> console.log(e.target)}
  placeholder="Contraseña"
/>
```
